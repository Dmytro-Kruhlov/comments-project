# 📌 Comments Project

A web application for creating and viewing comments with support for:

- nested comments (replies)
- captcha validation
- HTML content in comments
- pagination
- REST API (Django + DRF)

---

# 🚀 Tech Stack

- Python 3.14
- Django
- Django REST Framework
- PostgreSQL
- Docker / Docker Compose
- Pre-commit
